你可以通过以下三种方法使用React

使用托管在CDN上的React

我们在下载页面提供了托管在CDN上的版本。这些预编译的文件使用UMD格式化。通过<script>标签来把React和ReactDOM引入你的全局环境。It should also work out-of-the-box in CommonJS and AMD environments.
