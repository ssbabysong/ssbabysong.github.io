Virtualenv的使用

发表于 2016-08-02   |   分类于 Tools   |  

我个人使用的目的：便于不同版本python开发

安装:
$ sudo apt-get install python-virtualenv
建存放目录：
$ mkdir testvirtual
创建虚拟环境（python3下的）：
$ virtualenv -p python3 env1
启动虚拟环境：
$ cd env1
$ source ./bin/activate
退出虚拟环境：
$ deactivate
可以直接在虚拟环境运行pip等
$ pip install xxx
